# springboot-restapi-postgresql-hibernate-crud-example

### Tools and Technologies
- springboot 2.3.6 (dependencies: SpringWeb, PostgreSQL, Spring Data JPA)
- java11
- postgreSQL
- JPA
- restAPI


### 기능
- 전체 테이블 조회
- id 에 해당하는 employee 조회
- id 에 해당하는 employee 정보 update
- id로 employee delete
